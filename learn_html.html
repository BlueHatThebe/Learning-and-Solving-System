What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating webpages and web applications. It structures the content using elements enclosed in tags.

Basic HTML Syntax


<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

Headings in HTML

HTML offers 6 levels of headings, from <h1> (largest) to <h6> (smallest).


<h1>Main Title</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

Adding Links


<a href="https://www.example.com">Visit Example</a>

Inserting Images


<img src="image.jpg" alt="Description" width="300"/>

Creating Lists

There are two types of lists: ordered and unordered.


<ul>
  <li>Apples</li>
  <li>Oranges</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Tables


<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>20</td>
  </tr>
</table>
